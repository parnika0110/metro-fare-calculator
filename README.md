# 🚇 Metro Fare Calculator

A smart web application to calculate metro routes, fares, interchanges, and travel insights using graph algorithms.

---

## 🌟 Features

- 🔍 Station search with dropdown  
- 🧭 Finds all routes using graph traversal  
- 📍 Shortest route (minimum stops)  
- 🔁 Interchange detection  
- 💰 Fare calculation (peak / non-peak + card discounts)  
- 👥 Passenger-based pricing  
- 🗣️ Voice route guidance  
- 🧾 Ticket export  
- 🕒 Trip planner & monthly pass estimator  
- ⚡ Works offline (localStorage support)

---

## 🧠 Tech & Concepts

- Graph Data Structure  
- Depth First Search (DFS)  
- Modular Architecture (Services)  
- Node.js + Express  
- MongoDB (optional)

---

## 🚀 Run the Project

Frontend:
Open `frontend/index.html`

Backend:
cd backend  
node server.js
